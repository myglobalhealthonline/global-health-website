Admin Portal Wireframes v1 · Stage 4

Screen 1 — Login

Entry point. Single form, role checked server-side. SUPER_ADMIN goes to dashboard, plain admins land on country selector.

Global Health Admin

Sign in to manage clinics, doctors & services

Forgot password?

Restricted access. All activity is logged.

Decisions

  1. Uses User table with role IN (ADMIN, SUPER_ADMIN). JWT via jose, secure httpOnly cookie.
  2. Middleware in admin/middleware.ts redirects unauthenticated traffic to /login.
  3. "Forgot password" goes to a placeholder for v1 (manual reset by SUPER_ADMIN) — full email flow is post-launch.
  4. Every successful login writes to AdminAuditLog with action=login.

Decisions
