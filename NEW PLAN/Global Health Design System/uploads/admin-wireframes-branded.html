Brand decisions

  1. Forest gradient background evokes the brand palette directly — primary brand color is the first thing admins see.
  2. Lime "Sign in" button uses the high-energy accent for the single most important action on the page.
  3. Decorative radial glow in Olive at top-right adds depth without competing with the form.
  4. Auth: User + jose JWT, httpOnly secure cookie. Every login writes to AdminAuditLog.
